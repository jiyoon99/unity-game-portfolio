# Unity Game Portfolio

Unity를 공부하면서 완성한 게임 프로젝트를 정리한 포트폴리오 저장소입니다.

## Projects

| Project | Genre | Description | Play |
| --- | --- | --- | --- |
| VamGame | 2D action survival | 몰려오는 적을 피하고 무기와 아이템을 성장시키는 생존형 액션 게임 | `games/vamgame/Build/VamGame/vam.exe` |
| subakgame | Unity Windows build | Unity로 제작한 Windows 실행 빌드 게임 | `games/subakgame/Build/subakgame/subakgame.exe` |

## Repository Structure

```text
games/
  vamgame/
    README.md
    VamGame.unitypackage
    Build/VamGame/
  subakgame/
    README.md
    Build/subakgame/
```

## Notes

- 이 저장소는 포트폴리오 공개를 위한 정리본입니다.
- `VamGame`은 Unity package와 Windows build를 함께 포함합니다.
- `subakgame`은 현재 보관된 자료가 Windows build 형태라 실행 빌드 중심으로 정리했습니다.
- 각 게임은 Windows 환경에서 실행 파일을 열어 플레이할 수 있습니다.
